小楼一夜听春雨
======

本博客是由[造轮子工程师](http://github.com/mytharcher/SimpleGray)设计的jekyll站点主题，中文名称：**极简灰**,克隆而成。

利用这个模板可以帮助你在GitHub上快速的搭建一个极简风格的博客站点。

### 获得源代码

直接clone这个仓库到本地：

	$ clone git://github.com/mytharcher/SimpleGray.git your-repo-name
